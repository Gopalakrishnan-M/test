CSS Attribute Selectors

CSS Attribute selectors are used to select the HTML elements based on an attribute or attribute value

  • [attribute]

    • Select elements with a specified attribute

  • [attribute="value"]

    • Select elements with a specified attribute value

  • [attribute~="value"]

    • Select elements with a specified attribute value

  • [attribute|="value"]

    • Select elements with a specified attribute value

  • [attribute^="value"]

    • Select elements with a specified attribute value

  • [attribute$="value"]

    • Select elements with a specified attribute value

  • [attribute*="value"]

    • Select elements whose attribute value contain a specified value

    (This value does not have to be a whole word)
