CSS Attribute Selectors

CSS Attribute selectors are used to select the HTML elements based on an attribute or attribute value

  • [attribute]

    • Select elements with a specified attribute

  • [attribute="value"]

    • Select elements with a specified attribute value

  • [attribute~="value"]

    • Select elements with a specified attribute value

  • [attribute|="value"]

    • Select elements with a specified attribute value

    The [attribute|="value"] selector is used to select elements with the specified attribute starting with the specified value. The following example selects all elements with a title attribute value that begins with "animal": Note: The value has to be a whole word, either alone, like title="flower", or followed by a hyphen( - ), like title="flower-4"

  • [attribute^="value"]

    • Select elements with a specified attribute value

  • [attribute$="value"]

    • Select elements with a specified attribute value

  • [attribute*="value"]

    • Select elements whose attribute value contain a specified value

    (This value does not have to be a whole word)
